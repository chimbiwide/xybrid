---
title: 什么是 Xybrid
description: 面向移动端、桌面端和边缘端的本地 AI SDK
---

Xybrid 是一个**本地 AI SDK**，让你可以直接在移动和桌面应用中运行 ML 模型。添加语音转文字、文字转语音、大语言模型、视觉模型和嵌入模型功能 — 全部在用户设备上本地运行，无需依赖云端。

## 你可以构建什么

- **语音转文字** — 使用 Whisper 在设备端转录音频
- **文字转语音** — 使用 Kokoro 生成自然语音
- **大语言模型** — 在本地运行语言模型，用于对话、摘要等
- **视觉模型** — 在设备端进行图像分类和目标检测
- **嵌入模型** — 生成用于搜索和相似度的向量嵌入

将这些能力串联成多步流水线，完全在设备端运行，或在需要更强算力时选择性地路由到云端 API。

## 支持的平台

| 平台 | 可用 SDK |
|----------|---------------|
| iOS | Flutter, Swift (早期版本) |
| Android | Flutter, Kotlin (早期版本) |
| macOS | Flutter, Swift (早期版本), Unity |
| Linux | Flutter |
| Windows | Flutter (计划中) |
